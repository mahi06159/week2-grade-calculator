# Student Grade Calculator – Week 2

## Project Description
A Python program that processes multiple students’ marks, calculates grades with comments, stores data using lists, and displays class statistics.

## What I Learned
- Conditional Logic (if / elif / else)
- Lists and list operations
- For and while loops
- Input validation
- Error handling with try-except
- Functions and modular coding

## Features
✔ Multiple students processing  
✔ Grade calculation with comments  
✔ Class statistics (average, highest, lowest)  
✔ Search student feature  
✔ Save results to file  
✔ Menu-based system  
✔ Color-coded output

```bash
cd week2-grade-calculator
